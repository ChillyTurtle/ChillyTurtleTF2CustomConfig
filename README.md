# ChillyTurtleTF2CustomConfig
My TF2 configs.

- sv_cheats 1
- sv_allow_point_servercommand always

- Better network settings
- Disable mouse acceleration
- Optimized advanced multiplayer options for gameplay

- Null movement script

# binds.cfg
